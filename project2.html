Video User Guide

About the Project

This project was similar to the print user guide, except this time I screen recorded myself choosing a template on Canva and then added voice-over instructions. I then edited the video to make it as concise as possible while still completely explaining the process.

Skills Gained

I didn't expect to work on my video editing skills in this course, but this project definitely allowed me to practice cutting clips and editing volume to make a video great. I also gained skills in my speaking, as I had to ensure my voice was clear and slow while I recorded my voiceover. I had to consider plain language for this project as well, which is a skill I will likely use for the rest of my life.
