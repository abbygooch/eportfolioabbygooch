User Persona

About the User Persona

For this project, I created a persona for Jane Doe based on one of the participants in my Usability Tests. Similar to the User Story, I considered demographics and psychographics to create a realistic user persona.

Skills Gained

This project also helped me gain research skills, specifically in gathering qualitative and quantitative data through user interviews, surveys, usability tests, and analytics. I also considered inclusive design awareness, and ensured to create a persona that reflect diverse needs, ensuring accessibility and usability for all.
